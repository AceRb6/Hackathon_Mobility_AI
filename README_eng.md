# Hackathon_Mobility_AI

## Description
This mobile application allows managing incidents within the Mexico City Metro through a workflow between Station Chief and Regulator. Reports are stored and synchronized in real-time using Firebase Firestore.

## Features
- Station Chief creates the report.
- The report is saved in Firestore.
- Regulator checks the reports from the collection.
- Regulator completes the technical information.
- Firestore updates the document.
